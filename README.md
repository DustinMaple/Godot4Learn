## Demo2d

第一个项目。

### 主要知识点

* 节点基础
* 定时器（Timer）
* RigidbodyNode2d添加物理相关
* AnimatedSprite2D处理动画相关
* CollisionShape2D碰撞相关
* VisibleOnScreenNotifier2D 与屏幕相关，用来控制敌人离开屏幕后消失。
* **信号**
* **HUD**
* **项目组织结构**
* **脚本挂载位置**
* 在脚本中引用节点：
  * 从Scene中拖拽到脚本。
  * 使用find
* 纯色背景（ColorRect）
* AudioStreamPlayer 处理音乐相关。
* Path + PathFollow2D 用来处理从屏幕周围刷怪
* Marker2D 在场景中留下一个位置标记
